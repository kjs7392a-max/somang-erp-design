## 최종 DB 구조 (Supabase: bnlybtvdqyfxmbrcvcnv)

### ENUM 타입
```sql
draft_status:    pending | in_progress | approved | held | rejected
approval_action: pending | approved | held | rejected
```

### 테이블 구조

#### corporations
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK, default gen_random_uuid() |
| name | TEXT | NOT NULL UNIQUE |
| short_name | TEXT | NOT NULL |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

**현재 데이터:** `소망의료재단` (short_name: 소망)

#### profiles
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK, FK → auth.users(id) |
| corporation_id | UUID | NOT NULL, FK → corporations(id) |
| full_name | TEXT | NOT NULL |
| department | TEXT | NOT NULL |
| position | TEXT | NOT NULL |
| is_super_admin | BOOLEAN | NOT NULL DEFAULT false |
| employment_status | TEXT | NOT NULL DEFAULT 'active' |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

**현재 데이터:** 윤민주 (n001@somang.internal / 간호과 / 간호사 / active)

#### global_viewers
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK |
| profile_id | UUID | NOT NULL UNIQUE, FK → profiles(id) |
| note | TEXT | - |
| granted_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

#### drafts
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK |
| corporation_id | UUID | NOT NULL, FK → corporations(id) |
| drafter_id | UUID | NOT NULL, FK → profiles(id) |
| title | TEXT | NOT NULL |
| doc_type | TEXT | NOT NULL |
| status | draft_status | NOT NULL DEFAULT 'pending' |
| submitted_at | TIMESTAMPTZ | - |
| completed_at | TIMESTAMPTZ | - |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

#### document_contents
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK |
| draft_id | UUID | NOT NULL UNIQUE, FK → drafts(id) |
| body | JSONB | NOT NULL DEFAULT '{}' |
| attachments | JSONB | DEFAULT '[]' |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |
| updated_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

#### draft_approval_steps
| 컬럼 | 타입 | 제약 |
|---|---|---|
| id | UUID | PK |
| draft_id | UUID | NOT NULL, FK → drafts(id) |
| approver_id | UUID | NOT NULL, FK → profiles(id) |
| order_index | INTEGER | NOT NULL CHECK >= 1 |
| action | approval_action | NOT NULL DEFAULT 'pending' |
| comment | TEXT | - |
| acted_at | TIMESTAMPTZ | - |
| created_at | TIMESTAMPTZ | NOT NULL DEFAULT now() |

### RLS 헬퍼 함수
| 함수 | 역할 |
|---|---|
| `my_corporation_id()` | 현재 사용자의 corporation_id 반환 |
| `is_super_admin()` | profiles.is_super_admin 확인 |
| `is_global_viewer()` | global_viewers 테이블 존재 여부 확인 |
| `can_access_draft(uuid)` | 기안 접근 권한 확인 |
| `current_approver_id(uuid)` | 현재 결재 차례인 approver_id 반환 |
| `is_current_approver(uuid)` | 현재 사용자가 결재 차례인지 확인 |

### 트리거
| 트리거 | 대상 | 동작 |
|---|---|---|
| `trg_sync_draft_status` | draft_approval_steps (AFTER UPDATE) | 결재 action 변경 시 drafts.status 자동 동기화 |
| `trg_guard_super_admin` | profiles (BEFORE UPDATE) | is_super_admin 승격 방지 |
