# 2026-06-29 작업일지 — 소망 ERP

## 총무과 대시보드 전용 로그인 연결 (부서 공용계정)

### 무엇을
`/general`(총무과 대시보드)에 **전용 로그인 페이지**(`/general/login`)를 신설하고, 개인 ERP 로그인(`/login`, 사번·지문·앱잠금)과 분리했다. 로그인 방식은 **부서 공용계정**: 한글 부서명 아이디 + `부서명+1234`.

### 왜
미로그인으로 `/general` 접속 시 개인 ERP `/login`으로 튕기고, 개인 로그인은 끝나면 `/home`으로 가버려 대시보드에 도달하지 못했다. 부서 대시보드는 공용계정 로그인(설계 `dept-console-login`)을 쓰기로 확정.

### 변경 파일
1. **신규** `src/app/general/login/page.tsx` — 한글 부서명 아이디 → 고정 이메일 매핑(`총무`/`총무과` → `general@somang.internal`) 후 `signInWithPassword`. 성공 시 `/general` 이동. 지문/앱잠금 없음.
2. `src/app/general/layout.tsx` — `GeneralGuard`를 레이아웃에서 제거(`AuthProvider`만). 로그인 페이지가 가드에 걸려 무한 리다이렉트되는 것 방지.
3. `src/app/general/page.tsx` — `<GeneralGuard>`로 대시보드 본문만 감쌈.
4. `src/features/general-dashboard/components/GeneralGuard.tsx` — 미로그인 리다이렉트 대상 `/login` → `/general/login`.
